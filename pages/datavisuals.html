Evolution Chain Chart

This chart allows you to visualize the evolution chain of any specific Pokémon. By entering a Pokémon's name, the chart will display its evolutionary stages, showing how it evolves over time.

Here's how you can use the chart:

  • Hover over each Pokémon to see detailed stats like HP, Attack, Defense, and more.
  • Click on a Pokémon to highlight it along with its direct evolutions, making it easy to track its evolutionary line.

    Type Distribution Chart

    This chart visualizes the distribution of Pokémon types in a bubble format. Each bubble represents a Pokémon type, with the size of the bubble indicating how many Pokémon belong to that type.

    You can interact with the chart in the following ways:

    • Hover over a bubble to see more details about the type, including a sample of Pokémon that belong to it.
    • Click on a bubble to highlight it and enlarge its size. Clicking again will return it to its original size.
    • Filter types by using the checkboxes provided. Selecting a checkbox will display only the types you are interested in.

    This interactive chart allows you to explore the different Pokémon types and learn how common each type is in the Pokémon world.
